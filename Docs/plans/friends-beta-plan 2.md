# Friends Beta Plan

Last locked: March 19, 2026

## First-Beta Goal

Ship a narrow but real Friends surface that lets people deliberately connect and view one another's progress without turning AddOne into a full social network.

## Locked First-Beta Model

- Social profile is required before Friends access.
- Each device exposes one active rotatable share code.
- A viewer requests access by code.
- The owner approves the request.
- Approved viewers can browse the shared board live and read-only until revoked.

## First-Beta Surface

- complete social profile gate
- show your share code
- request another board by code
- approve or reject incoming requests
- browse approved boards and recent progress live

## Explicitly Deferred

- username search/discovery
- activity feed
- reactions
- comments or threaded discussion
- push notifications
- challenge groups

## Future Direction

Challenge groups remain a post-beta direction:

- shared goal across multiple people
- aggregated board brightness based on how many people completed the day
- optional communication around the shared challenge

## Execution Order

1. `T-009` profile identity model and account surface
2. `T-001` beta Friends surface and social floor
3. `T-013` challenge groups and shared board model
