# Generate this file on the broker host with:
#   node ./mosquitto/render-passwords.mjs --env-file ./.env
#
# The rendered file includes:
# - the dedicated gateway broker account from `MQTT_GATEWAY_USERNAME` / `MQTT_GATEWAY_PASSWORD`
# - one per-device account for every active row returned by `list_active_device_mqtt_credentials()`
#
# In the hosted VPS stack this file is normally kept up to date by
# `mosquitto/watch-passwords.mjs` inside the `broker-password-sync` sidecar.
# `../sync-passwords.sh` remains the manual fallback.
